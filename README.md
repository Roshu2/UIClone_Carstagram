	
## 프로젝트 명  
Carstagram By KTLO  

## 팀원 및 역할
-김규민: 전체총괄, UI제공, 댓글, 모달, 유저정보 불러오기, 회원가입 백엔드   
<br>
-이동현: 사진업로드 기능, 좋아요, 개인페이지 넘어갈 시 개인게시물만 불러오기  
<br>
-김진수: 팀장, 발표, ppt준비, 회원가입 프론트엔드, 백엔드(기본 회원가입api)  
<br>
-김민재: 로그인페이지 프론트엔드, 백엔드

## 기능  
1. 회원가입 페이지 
2. 로그인 페이지
3. 메인 피드 페이지
4. 개인 프로필 페이지  

## 못다한 기능
1. 팔로우, 팔로잉 기능
2. 프로필 편집
3. 개인 페이지 게시글 갯수 카운트
